## This demo app is item catalog page.

## backend

#### install mongodb
https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/

#### install express
```
$ cd backend/
$ npm install
$ npm i express-generator -g
$ express --view=pug backend
```
#### run backend server
```
$ DEBUG=backend:* npm start
```

## frontend

#### create react app
```
$ npx create-react-app 'myapp'
```
#### run frontend server
```
$ npm start
```
